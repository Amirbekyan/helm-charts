## Twenty CRM Helm Chart
This is a Helm chart for deploying [Twenty](https://github.com/twentyhq/twenty).

Chart supports:
* Deployment of:
  1. Redis standalone server
  2. PostgreSQL standalone server
  3. Twenty CRM server
  4. Twenty CRM worker
* Persistence
* Ingress
